Automatic Rain-Activated Wiper System

Embedded Systems Electronics Integrated Circuits

Designed a system that automatically activates a wiper mechanism when rain is detected using water sensors and integrated electronic circuits. The project focused on sensor integration, signal processing, and basic control logic.

FAST Robotics Competition — Autonomous Line Follower Robot

Robotics ESP32 PID Control Embedded Systems Competition

We participated in the FAST competition held at the Sidi Abdellah University Hub, where we designed and developed a fully autonomous line follower robot. The system was built using an ESP32 microcontroller and implemented a PID regulation algorithm to ensure precise trajectory tracking and stability. This project allowed us to practically apply control theory concepts studied in our engineering curriculum, particularly feedback systems, real-time correction, and parameter tuning. Through this challenge, we strengthened our skills in embedded systems, sensor integration, algorithm optimization, and teamwork under competitive conditions.
